# Contributing to MS 365 Skill

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch for your feature or bugfix
4. Make your changes
5. Run tests and ensure everything works
6. Submit a pull request

## Development Setup

1. Install required dependencies:
   ```bash
   npm install -g @softeria/ms-365-mcp-server
   ```

2. (Optional) For the Python CLI wrapper, ensure Python 3.8+ is installed
   - No additional Python packages required (uses standard library only)

## Contribution Guidelines

- Follow the existing code style
- Add tests for new features
- Update documentation
- Keep pull requests focused and concise

## Reporting Issues

- Use GitHub Issues
- Provide clear, reproducible steps
- Include error messages and context

## Code of Conduct

Be respectful, inclusive, and constructive in all interactions.
